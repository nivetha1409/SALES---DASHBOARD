## Project Overview  
- This Excel Dashboard provides an interactive analysis of sales data using Pivot Tables, Pivot Charts, and Slicers.   
- It helps users monitor sales performance, product trends, regional performance, and salesperson contributions through dynamic visualizations.  

## Objectives  
- Analyze total sales and profit performance.  
- Monitor product-wise sales and units sold.  
- Compare sales across different regions.  
- Evaluate salesperson performance.  
- Provide interactive filtering for better decision-making.
## Tools Used    
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers  
- Conditional Formatting  
- Dashboard Design Techniques
## Dashboard Features    
### 1. KPI Summary Cards  
**Displays key business metrics:**    
- Total Sales  
- Total Units Sold  
- Total Profit  
- Average Sales  
### 2. Region-wise Sales Analysis  
- Shows total sales generated in each region.  
- Helps identify top-performing regions.  
### 3. Product Performance Analysis  
- Compares sales across different products.  
- Identifies best-selling and low-performing products.  
### 4. Salesperson Performance  
- Displays sales contribution of each salesperson.  
- Helps evaluate individual performance.  
### 5. Units Sold Analysis  
- Tracks product-wise units sold.  
- Provides insights into customer demand.  
### 6. Interactive Slicers  
**Users can filter dashboard data by:**  
- Sales Person  
- Product  
- Region  
### 7. Visual Reports  
- Doughnut Chart for Region-wise Sales  
- Bar Chart for Product Sales  
- Column Chart for Salesperson Performance  
- Line Chart for Units Sold Analysis
## Benefits
- Easy-to-understand visual reports.
- Interactive filtering and analysis.
- Quick identification of sales trends.
- Supports data-driven business decisions.
